DOM - document object model

DOM - tai visas HTML turinys paverstas JavaScript objektu. Kiekvienas HTML kompiuterinės kalbos elementas yra paverčiamas JavaScript objektu ir susiejamas medžio hierarchija. Visas šis medžio hierarchija surestas objektų medis saugomas kintamajame document.

Keičiant kintamajį document (ar jame esančius elementus) JavaScript variklis incijuoja 'Render' variklio perpiešimo logiką. Tokiu būdu atsinaujina 'viewport'e matomas vaizdas.

Keičiant document (ar jame esančius elementus) galima atlikti tokius veikmus:

  • Kurti
  • Trinti
  • Redaguoti
  • Stilizuoti

Elementų paieška

Norint su JavaScript keisti HTML turinį (keisti JavaScript objektus, kurie susieti su HTML turiniu) turime surasti elementus. Tai darome ieškant elementų pagal selektorių. Selektoriaus logika yra tokia pati kaip ir CSS selektorių.

Element.querySelector - surasti pirmą elementą pagal selektorių

//  By attribute id=""
const main = document.querySelector('#main');
//  By attribute class=""
const h1 = main.querySelector('.h1');
//  By tag name 
const mainParagraph = main.querySelector('p');

Element.querySelectorAll - surasti visus elementus pagal selektorių

// Select many by criteria
const mainListItems = document.querySelectorAll('.js-mainListItem');
